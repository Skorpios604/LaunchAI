### Overview
This project is a Next.js landing page featuring a stunning WebGL animated shader background (transparently blending over the global app layout), glassmorphism UI elements, and a high-tech "cyberpunk" aesthetic.

## Development

First, run the development server locally:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment
This project is continuously deployed to GitHub Pages via GitHub Actions whenever changes are pushed to the `master` branch.
